# Requirement : 

write a Flowchart program to : 

ask the user to enter :

- Grade

Then Print the grade as follows :

- 90 - 100 Print A 
- 80 - 89 Print B 
- 70 - 79 Print C 
- 60 - 69 Print D 
- 50 -59 Print E
- otherwise Print F 

# Solution : 

```mermaid
graph TD

    A(Start) --> B[/Read Grade/]

    B --> C{Grade >= 90}

    C -->|Yes| D[/Print A/]

    C -->|No| E{Grade >= 80}

    E -->|Yes| F[/Print B/]

    E -->|No| G{Grade >= 70}

    G -->|Yes| H[/Print C/]

    G -->|No| I{Grade >= 60}

    I -->|Yes| J[/Print D/]

    I -->|No| K{Grade >= 50}

    K -->|Yes| L[/Print E/]

    K -->|No| M[/Print F/]

    D --> N(End)

    F --> N

    H --> N

    J --> N
```
