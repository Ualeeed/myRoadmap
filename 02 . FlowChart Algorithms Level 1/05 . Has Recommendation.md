# Requirement : 

write a Flowchart program to : 

ask the user to enter his/her :

- Age
- Driver license
- Has Recommendation!

Then Print "Hired" if his/her age is grater than 21 and she/he has a driver license, otherwise Print "Rejected" Or if he Has Recommendation Hire him/her without conditions!

# Solution : 

```mermaid
graph TD

    A(   Start   ) --> B[/Read Age, Read Has Driver License, Read Has Recommendation/]

    B --> C{Has Recommendation == True?}

    C -- No --> D[Check if Age is >= 21 and Has Driver License is True]

    D --> E{Is Result True?}

    E -- No --> F[/Print Rejected/]

    E -- Yes --> G[/Print Hired/]

    F --> H[(End)]

    G --> H[(End)]

    C -- Yes --> G
```
