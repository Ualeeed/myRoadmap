# Requirement : 

write a Flowchart program to : 

- read the ATM PIN code from the user , then check if PIN code = 1234,  then show the balance to user , otherwise print " Wrong PIN " and ask the user to enter the PIN again .

Only allow user to enter the PIN 3 times , if fails , print " Card is locked! "

# Solution : 

```mermaid
graph TD

    A(Start) --> B[FailedCount = 0]

    B --> C[/Read PIN/]

    C --> D{PIN = 1234?}

    D -->|Yes| E[Balance = 7500]

    E --> F[/Print Your Balance is:  + Balance/]

    F --> G(End)

    D -->|NO| H[/Print Wrong PIN/]

    H --> I[FailedCounter = FailedCounter +1]

    I --> K{FailedCounter = 3}

    K -->|NO| C

    K -->|Yes| Q[/Print Card locked/]

    Q --> G

    linkStyle 3 stroke:#38B6FF

    linkStyle 6 stroke:#FF3131

    linkStyle 9 stroke:#FF3131

    linkStyle 10 stroke:#38B6FF
	
```
