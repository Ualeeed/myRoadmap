# Requirement : 

write a Flowchart program to : 

calculate circle area then print it on the screen.


$$


{Area} = \pi \cdot r^2 



$$
the user should enter 
- r 

Area = PI * (r * r )

# Solution :  

```mermaid
graph TD

    A(Start) --> B[/Read r/]

    B --> C[PI = 3.14<br>Area = PI * r * r]

    C --> D[/Print Area/]

    D --> E(End)
```
