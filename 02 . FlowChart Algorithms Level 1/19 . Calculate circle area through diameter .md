# Requiremenet : 

write a Flowchart program to : 

calculate circle area through diameter then print it on the screen.

$$


{Area} = \frac{\pi \cdot D ^2}{4} 



$$

the user should enter 
- D 

Area = (PI * D * D) / 4 

# Solution :  

```mermaid
graph TD

    A(Start) --> B[/Read D/]

    B --> C[PI = 3.14<br>Area = PI * D * D / 4]

    C --> D[/Print Area/]

    D --> E(End)
```
