# Requirement : 

write a Flowchart program to : 

- read the ATM PIN code from the user , then check if PIN code = 1234, then show the balance to user , otherwise print " Wrong PIN " and ask the user to enter the PIN again.

# Solution : 

```mermaid
graph TD

    A(Start) --> B[/Read PIN/]

    B --> C{PIN = 1234?}

    C -->|Yes| D[Balance = 7500]

    D --> E[/Print Your Balance is:  + Balance/]

    C -->|No| F[/Print Wrong PIN/]

    F --> B

    E --> G(End)

  

    linkStyle 4 stroke:#FF3131

    linkStyle 2 stroke:#38B6FF

```
