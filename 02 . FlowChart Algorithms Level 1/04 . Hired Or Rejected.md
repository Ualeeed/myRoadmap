# Requirement : 

write a Flowchart program to : 

ask the user to enter his/her :

- Age
- Driver license

Then Print "Hired" if his/her age is grater than 21 **and** she/he has a driver license, otherwise Print "Rejected"

# Solution : 

```mermaid

flowchart TD
    Start([Start])
    ReadInputs[/"Read Age Read HasDriverLicense"/]
    CheckCondition["Result = (Age > 21 and HasDriverLicense = True)"]
    Decision{"Result = True?"}
    Hired[/"Print 'Hired'"/]
    Rejected[/"Print 'Rejected'"/]
    End([End])
    
    Start --> ReadInputs
    ReadInputs --> CheckCondition
    CheckCondition --> Decision
    Decision -->|Yes| Hired
    Decision -->|No| Rejected
    Hired --> End
    Rejected --> End
```
