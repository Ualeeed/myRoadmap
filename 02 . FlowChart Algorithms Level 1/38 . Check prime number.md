# Requirement : 

write a Flowchart program to : 

read a number and check if it is a prime number or not .

<mark style="background: #FFF3A3A6;">Note </mark>: Prime number can only divide on one and on itself.

# Solution :

```mermaid
graph TD

    A(Start) --> B[/Read N/]

    B --> C{N > 0?}

    C -->|Yes|D[counter = 2 , M = Round N/2]

    C -->|NO| E[/Print Num must be >0/]

    E --> B

    D --> F{N <= counter}

    F --> |Yes| G[/Print Prime/]

    F --> |NO| K{N Mod<br> counter = 0}

    K --> |Yes| L[/Print Not Prime/]

    K --> |No | J{Counter = M}

    J --> |Yes| G

    J --> |NO | M[Counter = Counter + 1]

    M --> K

    L --> O(End)

    G --> O

    linkStyle 2 stroke:#38B6FF

    linkStyle 3 stroke:#FF3131

    linkStyle 7 stroke:#FF3131

    linkStyle 6 stroke:#38B6FF

    linkStyle 8 stroke:#38B6FF

    linkStyle 9 stroke:#FF3131

    linkStyle 10 stroke:#38B6FF

    linkStyle 11 stroke:#FF3131
```
