# Requirement : 

write a Flowchart program to : 

calculate the task duration in seconds and print it on screen 

→ Giving the time duration of a task in the number of days, hours, minutes, and second. 

# Solution : 

```mermaid
graph TD

    A(Start) --> B[/Read <br> NumberOfDays,<br> NumberOfHours,<br> NumberOfMinutes,<br> NumberOfSeconds/]

    B --> C[TotalSeconds =<br> NumberOfDays * 24 * 60 * 60<br> + NumberOfHours * 60 * 60<br> + NumberOfMinutes * 60 <br>+ NumberOfSeconds]

    C --> D[/Print TotalSeconds +  Seconds/]

    D --> E(End)
```
