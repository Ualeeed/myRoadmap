# Requirement : 

write a Flowchart program to : 

ask the user to enter :

- Number 1 
- Number 2
- Operation type 

then perform the calculation according to the Operation type as following :

- " + " : add the two numbers.
- " - "  : Subtract the two numbers.
- " * " : Multiply the two numbers.
- " / " : Divide the two numbers. 

# Solution : 

```mermaid 
graph TD

    A[Start] --> B[Read Num1, Num2, OpType]

    B --> C{OpType = +}

    C -->|Yes| D[Print Num1 + Num2]

    C -->|No| E{OpType = -}

    E -->|Yes| F[Print Num1 - Num2]

    E -->|No| G{OpType = *}

    G -->|Yes| H[Print Num1 * Num2]

    G -->|No| I{OpType = / }

    I -->|Yes| J[Print Num1 / Num2]

    I -->|No| K[Print Wrong Operation]

    D --> L[End]

    F --> L

    H --> L

    J --> L

    K --> L
```
