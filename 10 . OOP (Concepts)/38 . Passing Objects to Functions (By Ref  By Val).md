##### <span style="color:rgb(0, 176, 80)">Objects</span> can be treated like any <span style="color:rgb(20, 192, 255)">data type</span> such as <span style="color:rgb(20, 192, 255)">int</span>, <span style="color:rgb(20, 192, 255)">bool</span>, <span style="color:rgb(0, 176, 80)">string</span> `...`,


## They can be passed to <span style="color:rgb(255, 255, 0)">functions</span> as parameters either by <span style="color:rgb(255, 66, 66)">reference</span> or by <span style="color:rgb(255, 66, 66)">value</span>.


##### Example : 


```cpp

#include<iostream>

using namespace std;

class clsA
{

public:
	int x;

	void Print()
	{
		cout << "The value of x=" << x <<endl;
	}

};


//object sent by value, any updated will not b reflected
// on the original object
void Fun1(clsA A1)
{

	A1.x = 100;
}


//object sent by reference, any updated will be reflected
// on the original object
void Fun2(clsA &A1)
{

	A1.x = 200;
}


int main() 

{
	clsA A1;

	A1.x = 50;
	cout << "\nA.x before calling function1: \n";
	A1.Print();


	//Pass by value, object will not be afected.
	Fun1(A1);
	cout << "\nA.x after calling function1 byval: \n";
	A1.Print();

	//Pass by ref, object will be afected.
	Fun2(A1);
	cout << "\nA.x after calling function2 byref: \n";
	A1.Print();


	system("pause>0");

}

```
