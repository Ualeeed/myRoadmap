### What is a Nested Class?

A **nested class** is simply a class defined **inside another class**.

##### Example : 

```cpp
class clsPerson {
    
    class clsAddress
    {
    public:
        string AddressLine1;
        string AddressLine2;
        string City;
        string Country;
      
        void Print()
        {
            cout << "\nAddress:\n";
            cout << AddressLine1 << endl;
            cout << AddressLine2 << endl;
            cout << City << endl;
            cout << Country << endl;
        }
    };

public:
    string FullName;
    clsAddress Address;
    ...
};
```

Here:

- `clsPerson` is the **outer class**.
    
- `clsAddress` is a **nested class** (defined inside `clsPerson`).


- Inside `clsPerson`, you declare `clsAddress Address;` → so every person object will automatically have an **Address object**.
    
### Why use nested classes?

✅ **Encapsulation**: `clsAddress` is meaningful only inside `clsPerson`.  
✅ **Organized code**: Keeps related classes grouped together.  
✅ **Cleaner design**: Instead of having too many separate classes in global scope, you group related ones.

#### Full Example : 

```cpp

#include<iostream>
using namespace std;

class clsPerson {
    
    class clsAddress
    {
    public:
        string AddressLine1;
        string AddressLine2;
        string City;
        string Country;
      
        void Print()
        {
            cout << "\nAddress:\n";
            cout << AddressLine1 << endl;
            cout << AddressLine2 << endl;
            cout << City << endl;
            cout << Country << endl;
        }
    };

public:
    string FullName;
    clsAddress Address;

    clsPerson()
    {
        FullName = "Mohammed Abu-Hadhoud";
        Address.AddressLine1 = "Building 10";
        Address.AddressLine2 = "Queen Rania Street";
        Address.City = "Amman";
        Address.Country = "Jordan";
    }
    

};

int main() 

{

    clsPerson Person1;
   
    Person1.Address.Print();

    system("pause>0");
    return 0;
}
```
