# **Objects** with Parameterized Constructor and <span style="color:rgb(0, 176, 80)">Array</span>


#### Example : 

```cpp
#include<iostream>
#include<vector>

using namespace std;

class clsA
{
public:

	//Parametarized Constructor
	clsA(int value)
	{
		x = value;
	}

	int x;

	void Print()
	{
		cout << "The value of x=" << x <<endl;
	}

};


int main() 

{
	
	// Initializing 3 array Objects with function calls of
    // parameterized constructor as elements of that array
	clsA obj[] = { clsA(10), clsA(20), clsA(30) };

	// using print method for each of three elements.
	for (int i = 0; i < 3; i++) {
		obj[i].Print();
	}

	return 0;

	
	system("pause>0");

}


```


---

### 1. Declaring the array with parameterized constructors

```c++
clsA obj[] = { clsA(10), clsA(20), clsA(30) };
```

- This creates an array of 3 objects of type `clsA`.
    
- Each element is **directly initialized** using the parameterized constructor:
    
    - `clsA(10)` → calls constructor, sets `x = 10`.
        
    - `clsA(20)` → calls constructor, sets `x = 20`.
        
    - `clsA(30)` → calls constructor, sets `x = 30`.
        

✅ Notice: **No default constructor is needed** here, because you’re explicitly giving values to each element at creation time.

---

### 2. What happens in memory?

- A contiguous block of 3 objects is allocated on the stack (not on the heap, because you didn’t use `new`).
    
- For each element, the constructor `clsA(int value)` is called once.
    
- After construction:
    
    ```
    obj[0].x = 10
    obj[1].x = 20
    obj[2].x = 30
    ```
    

---

### 3. Printing

```c++
for (int i = 0; i < 3; i++) {
    obj[i].Print();
}
```

This loops over all objects and prints their `x` values:

```
The value of x=10
The value of x=20
The value of x=30
```

---

### 4. Differences from your previous code

|Your first code (with `new`)|This new code|
|---|---|
|Used `new clsA[NumberOfObjects]` → objects created on **heap** with default constructor.|Uses `{ clsA(10), clsA(20), clsA(30) }` → objects created on **stack**, directly with parameterized constructor.|
|Needed a default constructor.|Doesn’t need a default constructor.|
|Needed manual cleanup with `delete[]`.|Automatic cleanup when `main()` ends.|
|First constructs with default constructor, then assigns values in a loop.|Direct initialization → constructor called only once.|

In this file, you’re creating an array of objects initialized **directly** with the parameterized constructor. It’s cleaner than your first version because there’s no extra assignment step and no memory management to worry about.
