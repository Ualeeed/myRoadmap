# **What are Static Members?**

In a class, **static members** are **shared by all objects of that class**, instead of each object having its own copy.

- Can be **variables** or **methods**
    
- **Belong to the class itself**, not to individual objects
    
- Accessed using **ClassName::Member** or via object (but class access is preferred)
- Static variable initialization outside the class.


|Feature|Static Members|Non-Static Members|
|---|---|---|
|Belongs to|Class|Object|
|Copies per class|1|Each object has its own|
|Access|ClassName::Member|Object.Member|
|Can access non-static?|❌ No|✅ Yes|
|Lifetime|Entire program execution|While object exists|


```cpp
 #include <iostream>

 #include <string>

 using namespace std;

  

class clsA

{

   public:

      int var;

      static int counter;

  

      clsA() // constratctor

      {

         counter++;

      }

  

      void PrintResult()

      {

         cout << "\nvar = " << var << endl;

         cout << "counter = " << counter << endl;

      }

};

  

int clsA::counter = 0; //static variable initialisation outside the class

  

 int main()

 {

   clsA A1, A2, A3;

  

   A1.var = 10;

   A2.var = 20;

   A3.var = 30;

  

   A1.PrintResult();

   A2.PrintResult();

   A3.PrintResult();

  

   A1.counter = 100;

  

   cout << "\nAfter changing the static member counter in one Object :\n";

  

   A1.PrintResult();

   A2.PrintResult();

   A3.PrintResult();

  

    system("pause>0");

    return 0;

 }
```
