# Requirement : 

- Write a program that print all the form from AAA to ZZZ

```ad-example
AAA . AAB . AAC . AAD ... ZZZ

```

# Answer : 

```cpp
#include <iostream>
#include <string>
using namespace std;

void PrintAllFromAAAtoZZZ()
{
    cout << "\n";
    string word = "";

    for (int i = 65; i <= 90; i++)
    {
        for (int j = 65; j <= 90; j++)
        {
            for (int k = 65; k <= 90; k++)
            {
                word.append(1, char(i));
                word.append(1, char(j));
                word.append(1, char(k));
                cout << word << endl;
                word = "";
            }
        }
        cout << "\n____________________________\n";
    }
}
int main()
{
    PrintAllFromAAAtoZZZ();
    return 0;
}

```
