bool IsIdentityMarix(int Matrix1[3][3], short Rows, short Cols)
{
	//check Diagonal elements are 1 and rest elements are 0
	for (short i = 0; i < Rows; i++)
	{
		for (short j = 0; j < Cols; j++)
		{
			//check for diagonals element
			if (i == j && Matrix1[i][j] != 1)
			{
				return false;
			}
			//check for rest elements
			else if (i != j && Matrix1[i][j] != 0)
			{
				return false;
			}
		}
	}
	return true;
}

## Step-by-Step Algorithm Solution

### Algorithm: Check if Matrix is Identity Matrix

**Problem:** Determine if a given matrix is an identity matrix (diagonal elements are 1, all other elements are 0).

**Steps:**
1. **Start** the algorithm
2. **Input:** Accept matrix dimensions (rows and columns) and matrix elements
3. **Initialize** result as true
4. **For each row i** from 0 to rows-1:
   - **For each column j** from 0 to columns-1:
     - **If** i equals j (diagonal element):
       - **If** matrix[i][j] is not equal to 1:
         - **Set** result to false
         - **Break** from loops
     - **Else** (non-diagonal element):
       - **If** matrix[i][j] is not equal to 0:
         - **Set** result to false
         - **Break** from loops
5. **Output** the result (true if identity matrix, false otherwise)
6. **End** the algorithm

**Time Complexity:** O(nÂ²) where n is the dimension of the square matrix
**Space Complexity:** O(1) - only using a few variables

**Key Points:**
- An identity matrix must be square (rows = columns)
- Diagonal elements (where i == j) must be 1
- All non-diagonal elements (where i != j) must be 0
- Early termination when any condition fails improves efficiency
