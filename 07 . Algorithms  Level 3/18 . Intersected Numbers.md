```cpp
#include <iostream>
#include <string>
#include <iomanip>

using namespace std;


void PrintMatrix(int arr[3][3], short Rows, short Cols)
{
	for (short i = 0; i < Rows; i++)
	{
		for (short j = 0; j < Cols; j++)
		{
			//printf(" %0*d ", 2, arr[i][j]);
			cout << setw(3) << arr[i][j] << " ";
		}
		cout << "\n";
	}
}


bool IsNumberInMatrix(int Matrix2[3][3], int Number, short Rows,
	short Cols)
{
	
	for (short i = 0; i < Rows; i++)
	{
		for (short j = 0; j < Cols; j++)
		{
			if (Matrix2[i][j] == Number)
			{
				return true;
			};
		}
	}
	return false;
}


void PrintIntersectedNumbers(int Matrix1[3][3], int Matrix2[3][3],
	short Rows, short Cols)
{
	int Number;
	for (short i = 0; i < Rows; i++)
	{
		for (short j = 0; j < Cols; j++)
		{
			Number = Matrix1[i][j];
			if (IsNumberInMatrix(Matrix2, Number, Rows, Cols))
			{
				cout << setw(3) << Number << " ";
			}
		}
	}
}


int main()
{

	int Matrix1[3][3] = { {77,5,12},{22,20,1},{1,0,9} };

	int Matrix2[3][3] = { {5,80,90},{22,77,1},{10,8,33} };


	cout << "\nMatrix1:\n";
	PrintMatrix(Matrix1, 3, 3);

	cout << "\nMatrix2:\n";
	PrintMatrix(Matrix2, 3, 3);


	cout << "\nIntersected Numbers are: \n\n";

	PrintIntersectedNumbers(Matrix1, Matrix2, 3, 3);



	system("pause>0");
}

```



## Step-by-Step Algorithm Solution

### Algorithm: Find Intersected Numbers in Two Matrices

**Problem:** Find and print all numbers that exist in both matrices (intersection of two matrices).

**Steps:**
1. **Start** the algorithm
2. **Input:** 
   - Accept dimensions for both matrices (rows and columns)
   - Accept elements for Matrix1 and Matrix2
3. **Initialize** output message
4. **For each row i** from 0 to rows-1 in Matrix1:
   - **For each column j** from 0 to columns-1 in Matrix1:
     - **Get** current number from Matrix1[i][j]
     - **Check** if this number exists in Matrix2:
       - **Call** IsNumberInMatrix function with Matrix2 and current number
       - **If** the number is found in Matrix2:
         - **Print** the intersected number
5. **End** the algorithm

**Helper Function - IsNumberInMatrix:**
1. **Input:** Matrix, target number, rows, columns
2. **For each element** in the matrix:
   - **If** element equals target number:
     - **Return** true
3. **If** no match found:
   - **Return** false

**Time Complexity:** O(n²×m²) where n×n and m×m are the matrix dimensions
**Space Complexity:** O(1) - only using temporary variables

**Key Points:**
- Algorithm finds common elements between two matrices
- May print duplicate intersections if a number appears multiple times in Matrix1
- Order of output follows the order of elements in Matrix1
- Can be optimized using hash sets for better performance
- Works with matrices of any size (not limited to square matrices)

**Example:**
- Matrix1: [[77,5,12], [22,20,1], [1,0,9]]
- Matrix2: [[5,80,90], [22,77,1], [10,8,33]]
- Intersected: 77, 5, 22, 1, 1 (note: 1 appears twice in Matrix1)

**Optimization Note:** For large matrices, consider using a hash set to store Matrix2 elements for O(1) lookup time.
