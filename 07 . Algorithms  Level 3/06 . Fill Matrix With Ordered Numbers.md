#### Requirement : 

- Write a program that fill matrix with ordered numbers

#### Solution : 


```cpp
#include <iostream>
#include <string>

using namespace std;


void FillMatrixWithOrderedNumbers(int arr[3][3], short Rows, short
	Cols)
{
	short Counter = 0;

	for (short i = 0; i < Rows; i++)
	{
		for (short j = 0; j < Cols; j++)
		{
			Counter++;

			arr[i][j] = Counter;
		}
	}
}
void PrintMatrix(int arr[3][3], short Rows, short Cols)
{
	for (short i = 0; i < Rows; i++)
	{
		for (short j = 0; j < Cols; j++)
		{
			cout << arr[i][j] << " ";
		}
		cout << "\n";
	}
}


int main()
{
	int arr[3][3];

	FillMatrixWithOrderedNumbers(arr, 3, 3);

	cout << "\nThe following is a 3x3 ordered matrix:\n";

	PrintMatrix(arr, 3, 3);


	system("pause>0");
}

```



#### Code Explanation & Algorithm Flow:

**Algorithm Flow:**
1. **Matrix Declaration**: Create a 3x3 integer array
2. **Sequential Filling**: Fill matrix with ordered numbers starting from 1
3. **Matrix Display**: Print the ordered matrix

**How the Code Works:**

1. **`FillMatrixWithOrderedNumbers(int arr[3][3], short Rows, short Cols)`**:
   - **Key Innovation**: Uses a counter to assign sequential numbers
   - Initializes `Counter = 0` before filling begins
   - Uses nested loops to traverse matrix row by row
   - **Critical Step**: `Counter++` increments before assignment
   - Assigns current counter value: `arr[i][j] = Counter`
   - **Fill Pattern**: Row-major order (fills row 0 completely, then row 1, then row 2)

2. **`PrintMatrix(int arr[3][3], short Rows, short Cols)`**:
   - Standard matrix display function
   - Nested loops for row-by-row printing
   - Simple formatting without `setw()` for clean integer display

**Key Algorithm Concepts:**
- **Sequential Counter**: Single counter variable tracks current number to assign
- **Row-Major Traversal**: Natural nested loop order (rows outer, columns inner)
- **Incremental Assignment**: Counter increments with each position
- **Predictable Pattern**: Results in matrix filled 1,2,3... left-to-right, top-to-bottom

**Resulting Matrix Pattern:**
```
1 2 3
4 5 6
7 8 9
```

**Algorithm Efficiency:**
- **Time Complexity**: O(n√óm) where n=rows, m=columns
- **Space Complexity**: O(1) additional space (just counter variable)
- **Single Pass**: Each matrix element is visited and assigned exactly once

**Simplicity Advantage**: 
- No random number generation needed
- No external dependencies
- Deterministic, predictable output
- Easy to verify correctness
