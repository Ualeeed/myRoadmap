- Write a program to <span style="color:rgb(255, 31, 31)">check if the matrix is a scalar or not</span> : 


```cpp
#include <iostream>
#include <string>
#include <iomanip>

using namespace std;


void PrintMatrix(int arr[3][3], short Rows, short Cols)
{
	for (short i = 0; i < Rows; i++)
	{
		for (short j = 0; j < Cols; j++)
		{
			//printf(" %0*d ", 2, arr[i][j]);
			cout << setw(3) << arr[i][j] << " ";
		}

		cout << "\n";
	}
}

bool IsScalarMatrix(int Matrix1[3][3], short Rows, short Cols)
{
	int FirstDiagElemement = Matrix1[0][0];

	for (short i = 0; i < Rows; i++)
	{
		for (short j = 0; j < Cols; j++)
		{
			//check for diagonals element
			if (i == j && Matrix1[i][j] != FirstDiagElemement)
			{
				return false;
			}
			//check for rest elements
			else if (i != j && Matrix1[i][j] != 0)
			{
				return false;
			}
		}
	}
	return true;
}


int main()
{

	int Matrix1[3][3] = { {9,0,0},{0,9,0},{0,0,9} };


	cout << "\nMatrix1:\n";

	PrintMatrix(Matrix1, 3, 3);

	if (IsScalarMatrix(Matrix1, 3, 3))
		cout << "\nYES: Matrix is scalar.";
	else
		cout << "\nNo: Matrix is NOT scalar.";


	system("pause>0");
}

```



## Step-by-Step Algorithm Solution

### Algorithm: Check if Matrix is Scalar Matrix

**Problem:** Determine if a given matrix is a scalar matrix (all diagonal elements are the same non-zero value, all other elements are 0).

**Steps:**
1. **Start** the algorithm
2. **Input:** Accept matrix dimensions (rows and columns) and matrix elements
3. **Store** the first diagonal element as reference value: `firstDiagonal = matrix[0][0]`
4. **Initialize** result as true
5. **For each row i** from 0 to rows-1:
   - **For each column j** from 0 to columns-1:
     - **If** i equals j (diagonal element):
       - **If** matrix[i][j] is not equal to firstDiagonal:
         - **Set** result to false
         - **Break** from loops
     - **Else** (non-diagonal element):
       - **If** matrix[i][j] is not equal to 0:
         - **Set** result to false
         - **Break** from loops
6. **Output** the result (true if scalar matrix, false otherwise)
7. **End** the algorithm

**Time Complexity:** O(nÂ²) where n is the dimension of the square matrix
**Space Complexity:** O(1) - only using a few variables

**Key Points:**
- A scalar matrix is a special type of diagonal matrix
- All diagonal elements must have the same value
- All non-diagonal elements must be 0
- The scalar value (diagonal elements) can be any number (including 0)
- If all diagonal elements are 1, it becomes an identity matrix
