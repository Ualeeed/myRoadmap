```cpp
#include <iostream>
#include <string>
#include <iomanip>

using namespace std;


int RandomNumber(int From, int To)
{
	//Function to generate a random number
	int randNum = rand() % (To - From + 1) + From;
	return randNum;
}


void FillMatrixWithRandomNumbers(int arr[3][3], short Rows, short
	Cols)
{
	for (short i = 0; i < Rows; i++)
	{
		for (short j = 0; j < Cols; j++)
		{
			arr[i][j] = RandomNumber(1, 10);
		}
	}
}
void PrintMatrix(int arr[3][3], short Rows, short Cols)
{
	for (short i = 0; i < Rows; i++)
	{
		for (short j = 0; j < Cols; j++)
		{
			printf(" %0*d ", 2, arr[i][j]);
			//cout << setw(3) << arr[i][j] << " ";
		}
		cout << "\n";
	}
}


int SumOfMatrix(int Matrix1[3][3], short Rows, short Cols)
{
	int Sum = 0;
	for (short i = 0; i < Rows; i++)
	{
		for (short j = 0; j < Cols; j++)
		{
			Sum += Matrix1[i][j];
		}
	}
	return Sum;
}


bool AreEqualMatrices(int Matrix1[3][3], int Matrix2[3][3], short
	Rows, short Cols)
{
	return (SumOfMatrix(Matrix1, Rows, Cols) ==
		SumOfMatrix(Matrix2, Rows, Cols));
}


int main()
{
	//Seeds the random number generator in C++, called only once
	srand((unsigned)time(NULL));


	int Matrix1[3][3], Matrix2[3][3];

	FillMatrixWithRandomNumbers(Matrix1, 3, 3);

	cout << "\nMatrix1:\n";
	PrintMatrix(Matrix1, 3, 3);


	FillMatrixWithRandomNumbers(Matrix2, 3, 3);

	cout << "\nMatrix2:\n";
	PrintMatrix(Matrix2, 3, 3);


	if (AreEqualMatrices(Matrix1, Matrix2, 3, 3))
		cout << "\nYES: both martices are equal.";
	else
		cout << "\nNo: martices are NOT equal.";


	system("pause>0");
}

## Explanation

This program compares two matrices by checking if their sums are equal, **not by comparing individual elements**.

### Key Concepts:

**Matrix Equality by Sum:**
- This implementation defines "equal matrices" as matrices with equal sums
- **Important Note:** This is NOT the standard mathematical definition of matrix equality
- Standard matrix equality requires each corresponding element to be equal: A[i][j] = B[i][j] for all i,j

### Algorithm Breakdown:

1. **Matrix Generation:** Creates two separate 3x3 matrices with random numbers

2. **Sum-based Comparison (`AreEqualMatrices`):**
   - Calculates sum of first matrix using `SumOfMatrix()`
   - Calculates sum of second matrix using `SumOfMatrix()`
   - Returns `true` if both sums are equal, `false` otherwise

3. **Decision Logic:** Uses the boolean result to display appropriate message

### Example Scenarios:

**Scenario 1 - Same Sum, Different Elements:**
```
Matrix1:        Matrix2:
 5  1  4         2  3  5
 2  3  0         1  4  0  
 1  2  2         3  1  1

Sum1 = 20       Sum2 = 20
Result: "Equal" (but elements are different!)
```

**Scenario 2 - Different Sums:**
```
Matrix1:        Matrix2:
 5  1  4         8  3  5
 2  3  0         1  4  2  
 1  2  2         3  1  1

Sum1 = 20       Sum2 = 28
Result: "Not Equal"
```

### Mathematical Limitation:
This approach can produce false positives where matrices are considered "equal" despite having completely different element arrangements, as long as their sums match.

### Real-world Applications:
- Quick comparison when only total values matter
- Checksum validation for data integrity
- Statistical comparison of datasets

### Time Complexity: O(m√ón) for calculating both sums
### Space Complexity: O(1) - only storing sum values
