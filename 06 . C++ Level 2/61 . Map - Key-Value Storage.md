##  What is a Map?

A map in C++ is an associative container that stores elements as <span style="color:rgb(20, 192, 255)">key-value</span> pairs, where each key is unique and is used to identify and access its corresponding value. Maps automatically keep their elements sorted by keys.

Think of a map like a **real dictionary**:
- **Key** = The word you're looking up (e.g., "apple")
- **Value** = The definition of that word (e.g., "a red fruit")

### 2. **Inserting Elements**

There are multiple ways to add elements to a map:

```cpp
map<int, string> grades;

// Method 1: Using insert()
grades.insert({1, "A+"});
grades.insert(make_pair(2, "B+"));

// Method 2: Using [] operator (easier!)
grades[3] = "C+";
grades[4] = "A";

// Visual:
// grades â†’ {1:"A+", 2:"B+", 3:"C+", 4:"A"}
```

### 3. **Accessing Elements**

```cpp
map<string, int> prices = {
    {"Apple", 5},
    {"Banana", 3},
    {"Orange", 7}
};

// Method 1: Using []
cout << prices["Apple"];  // Output: 5

// Method 2: Using at() - safer!
cout << prices.at("Banana");  // Output: 3

// âš ï¸ Difference:
// [] creates new element if key doesn't exist
// at() throws error if key doesn't exist
```

### 4. **Finding Elements**

The find() function checks if a key exists in the map and returns an iterator pointing to it, or end() if not found.

```cpp
map<int, string> students = {
    {1, "Alice"},
    {2, "Bob"},
    {3, "Charlie"}
};

auto it = students.find(2);

if (it != students.end()) {
    cout << "Found: " << it->second;  // Output: Bob
} else {
    cout << "Not found!";
}
```

### 6. **Updating Values**

```cpp
map<string, int> scores = {
    {"Player1", 100},
    {"Player2", 150}
};

// Update existing value
scores["Player1"] = 200;      // Now 200
scores.at("Player2") = 175;   // Now 175

// Visual:
// Before: {"Player1":100, "Player2":150}
// After:  {"Player1":200, "Player2":175}
```

### 7. **Deleting Elements**

```cpp
map<int, string> data = {
    {1, "One"},
    {2, "Two"},
    {3, "Three"}
};

// Delete by key
data.erase(2);

// Delete all elements
data.clear();

// Check size
cout << data.size();  // Output: 0
```

##  Practical Example: Student Grade System

```cpp
#include <iostream>
#include <map>
#include <string>
using namespace std;

int main() {
    // Store student grades
    map<string, char> grades;
    
    // Add students
    grades["Alice"] = 'A';
    grades["Bob"] = 'B';
    grades["Charlie"] = 'A';
    grades["Diana"] = 'C';
    
    // Check if student exists
    string searchName = "Bob";
    if (grades.find(searchName) != grades.end()) {
        cout << searchName << "'s grade: " << grades[searchName] << endl;
    }
    
    // Display all grades
    cout << "\nAll Students:\n";
    for (auto& student : grades) {
        cout << student.first << " â†’ " << student.second << endl;
    }
    
    // Update grade
    grades["Bob"] = 'A';
    cout << "\nBob's new grade: " << grades["Bob"] << endl;
    
    return 0;
}
```
