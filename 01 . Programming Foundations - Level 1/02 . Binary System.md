 " How to represent other languages like Arabic for example ? " 
 
- in this case you need to use <span style="color:rgb(0, 176, 240)">UNICODE</span>.

- <span style="color:rgb(0, 176, 240)">UNICODE</span> is an International character encoding standards
- <span style="color:rgb(0, 176, 240)">UNICODE</span> has three standards : 

1. UTF-8 = 256 characters
2. UTF-16 = 65,536 characters 
3. UTF-32 = 4,294,967,296 characters
