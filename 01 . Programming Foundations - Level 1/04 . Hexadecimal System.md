" How to convert from Decimal to Hexadecimal ? "

Example : 469 to Hex 

| Number/16 | Result  | Integer | Fraction | Remainder*16        |
| --------- | ------- | ------- | -------- | ------------------- |
| 469/16    | 29.3125 | 29      | 0.3125   | 0.3125 * 16<br> = 5 |
| 29/16     | 1.8125  | 1       | 0.8125   | 0.8125 * 16<br>= 13 |
| 1/16      | --      | --      | --       | 1                   |

5 → in Hex is : 5
13 → in Hex is : D
1 → in Hex is : 1
(the way we read is down to up)

469 in Hexadecimal is : 1D5

  " How to covert <span style="color:rgb(0, 176, 80)">Hexadecimal</span> to <span style="color:rgb(181, 75, 251)">decimal</span>"
- Example : 1D5 
```

Position:  2   1   0
Hex:       1   D   5
Power:    16² 16¹ 16⁰
 
 5  *  16^0 = ( 5 * 1 ) = 5   
 D  *  16^1 = ( 13 * 16 ) = 208
 1  *  16^2 = ( 1 * 256 ) = 256 


5 + 208 + 256 = 469

```
