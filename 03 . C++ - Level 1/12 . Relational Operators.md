##  What are Relational Operators?

Relational operators are used to compare two values or expressions, and based on this comparison, it returns a ``boolean`` value (either true or false) as the result. Generally <mark style="background: #D33AFFA6;">false is represented as 0</mark> and <mark style="background: #1F52A5;">true is represented as any non-zero value</mark> (mostly 1).

When you need to compare values in your programs - checking if one number is bigger than another, or if two values are equal - relational operators are your essential tools.

| Operator | Name                  | Description                                   | Example  | Result     |
| -------- | --------------------- | --------------------------------------------- | -------- | ---------- |
| `>`      | Greater than          | Returns true if left operand > right operand  | `5 > 3`  | `true (1)` |
| `<`      | Less than             | Returns true if left operand < right operand  | `2 < 8`  | `true (1)` |
| `>=`     | Greater than or equal | Returns true if left operand >= right operand | `5 >= 5` | `true (1)` |
| `<=`     | Less than or equal    | Returns true if left operand <= right operand | `3 <= 7` | `true (1)` |
| ==       | Equal to              | Returns true if both operands are equal       | `4 == 4` | `true (1)` |
| `!=`     | Not equal to          | Returns true if operands are different        | `5 != 3` | `true (1)` |

##  Detailed Examples

### 1. Greater Than ( > )
The greater than ( > ) operator returns true if the left operand is greater than the right operand, otherwise, it returns false.

```cpp
#include <iostream>
using namespace std;

int main() {
    int score1 = 85;
    int score2 = 78;
    
    cout << \"Is score1 greater than score2? \" << (score1 > score2) << endl;  // Output: 1 (true)
    cout << \"Is 10 > 15? \" << (10 > 15) << endl;  // Output: 0 (false)
    cout << \"Is 25 > 25? \" << (25 > 25) << endl;  // Output: 0 (false)
    
    return 0;
}
```

### 2. Less Than ( < )  
```cpp
#include <iostream>
using namespace std;

int main() {
    int age = 16;
    int votingAge = 18;
    
    cout << \"Is age less than voting age? \" << (age < votingAge) << endl;  // Output: 1 (true)
    cout << \"Is 20 < 15? \" << (20 < 15) << endl;  // Output: 0 (false)
    
    return 0;
}
```

### 3. Greater Than or Equal ( >= )
```cpp
#include <iostream>
using namespace std;

int main() {
    int passingGrade = 60;
    int studentGrade = 75;
    
    cout << \"Did student pass? \" << (studentGrade >= passingGrade) << endl;  // Output: 1 (true)
    cout << \"Is 50 >= 50? \" << (50 >= 50) << endl;  // Output: 1 (true)
    cout << \"Is 40 >= 60? \" << (40 >= 60) << endl;  // Output: 0 (false)
    
    return 0;
}
```

### 4. Less Than or Equal ( <= )
```cpp
#include <iostream>
using namespace std;

int main() {
    int maxCapacity = 100;
    int currentPeople = 85;
    
    cout << \"Is within capacity? \" << (currentPeople <= maxCapacity) << endl;  // Output: 1 (true)
    cout << \"Is 15 <= 15? \" << (15 <= 15) << endl;  // Output: 1 (true)
    cout << \"Is 90 <= 70? \" << (90 <= 70) << endl;  // Output: 0 (false)
    
    return 0;
}
```

### 5. Equal To ( == )

### 6. Not Equal To  ( != )
```cpp

using namespace std;

int main() {

```

## âš ï¸ Important Notes and Best Practices

### 1. Return Values
- **True** is represented as `1` (or any non-zero value)
- **False** is represented as 0

### 2. Common Mistakes

**Mistake 1: Confusing = and == 

```cpp
// WRONG:
if (x = 10)  // This assigns 10 to x, doesn't compare!

// CORRECT:
if (x == 10)  // This compares x with 10
```

**Mistake 2: Chaining comparisons incorrectly**
```cpp
// WRONG:
if (10 < x < 20)  // This doesn't work as expected!

// CORRECT:
if (x > 10 && x < 20)  // Use logical AND
```

### 3. Floating-Point Comparisons
When the less-than (<), greater-than (>), less-than-equals (<=), and greater-than-equals (>=) operators are used with floating point values, they will produce a reliable answer in most cases (when the value of the operands is not similar). However, if the operands are almost identical, these operators should be considered unreliable.

```cpp
#include <iostream>
using namespace std;

int main() {
    double a = 0.1 + 0.2;  // May not exactly equal 0.3 due to floating-point precision
    double b = 0.3;
    
    cout << "a = " << a << endl;
    cout << "b = " << b << endl;
    cout << "a == b: " << (a == b) << endl;  // May output 0 (false)!
    
    // Better approach for floating-point comparison:
    double epsilon = 0.0001;
    cout << "Are they close enough? " << (abs(a - b) < epsilon) << endl;
    
    return 0;
}
```

### 4. Using with Different Data Types

```cpp
#include <iostream>
using namespace std;

int main() {
    // Character comparisons (ASCII values)
    char letter1 = 'A';  // ASCII value: 65
    char letter2 = 'B';  // ASCII value: 66
    
    cout << "'A' < 'B': " << (letter1 < letter2) << endl;  // Output: 1 (true)
    cout << "'Z' > 'A': " << ('Z' > 'A') << endl;          // Output: 1 (true)
    
    // Mixed type comparisons (automatic conversion)
    int intNum = 10;
    double doubleNum = 10.5;
    
    cout << "10 < 10.5: " << (intNum < doubleNum) << endl;  // Output: 1 (true)
    
    return 0;
}
```
