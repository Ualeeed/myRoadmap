#  Arithmetic Operators in C++

<mark style="background: #FFB86CA6;">Arithmetic operators</mark> are <mark style="background: #FF5582A6;">symbols that perform mathematical operations</mark> on variables and values. They are the foundation of mathematical calculations in C++ programming.

Think of them as the basic calculator functions you use every day! üßÆ

---

##  The 5 Basic Arithmetic Operators

| **Operator** | **Name** | **Description** | **Example** | **Result** |
|--------------|----------|-----------------|-------------|------------|
| `+` | **Addition** | Adds two numbers together | `10 + 5` | `15` |
| `-` | **Subtraction** | Subtracts second number from first | `10 - 5` | `5` |
| `*` | **Multiplication** | Multiplies two numbers | `10 * 5` | `50` |
| `/` | **Division** | Divides first number by second | `10 / 5` | `2` |
| `%` | **Modulo** | Returns the remainder of division | `10 % 3` | `1` |

---

##  Let's See Them in Action!

### **Basic Example:**
```c++
#include <iostream>
using namespace std;

int main() {
    int a = 10, b = 5;
    
    cout << "a = " << a << ", b = " << b << endl << endl;
    
    cout << "Addition:       a + b = " << a + b << endl;      // 15
    cout << "Subtraction:    a - b = " << a - b << endl;      // 5
    cout << "Multiplication: a * b = " << a * b << endl;      // 50
    cout << "Division:       a / b = " << a / b << endl;      // 2
    cout << "Modulo:         a % b = " << a % b << endl;      // 0
    
    return 0;
}
```

**Output:**
```
a = 10, b = 5

Addition:       a + b = 15
Subtraction:    a - b = 5
Multiplication: a * b = 50
Division:       a / b = 2
Modulo:         a % b = 0
```

##  Understanding Each Operator

### 1 . **Addition Operator (+)**
- **What it does:** Adds two numbers together
- **Real-world example:** Adding your age and your friend's age

```c++
int myAge = 20;
int friendAge = 25;
int totalAge = myAge + friendAge;  // 45
cout << "Combined age: " << totalAge << " years";
```

### 2 . **Subtraction Operator (-)**
- **What it does:** Takes away one number from another
- **Real-world example:** Finding the difference in prices

```c++
int originalPrice = 100;
int discount = 20;
int finalPrice = originalPrice - discount;  // 80
cout << "Final price: $" << finalPrice;
```

### 3 . **Multiplication Operator ( * )** 
- **What it does:** Multiplies two numbers
- **Real-world example:** Calculating total cost for multiple items

```c++
int itemPrice = 15;
int quantity = 4;
int totalCost = itemPrice * quantity;  // 60
cout << "Total cost: $" << totalCost;
```

### 4 . **Division Operator (/)**
- **What it does:** Divides one number by another
- **Two types of division:**

#### **Integer Division** (when both numbers are integers):
```c++
int total = 10;
int people = 3;
int perPerson = total / people;  // Result: 3 (not 3.33!)
cout << "Each person gets: " << perPerson << " items";
```
‚ö†Ô∏è **Important:** Integer division discards the decimal part!

#### **Floating-Point Division** (when at least one number has decimals):
```c++
double total = 10.0;
int people = 3;
double perPerson = total / people;  // Result: 3.33333
cout << "Each person gets: " << perPerson << " items";
```

### 5 . **Modulo Operator (%) - The Remainder Finder**
- **What it does:** Finds the remainder after division
- **Real-world example:** Checking if a number is even or odd

```c++
int number = 17;
int remainder = number % 2;  // 17 √∑ 2 = 8 remainder 1
if (remainder == 0) {
    cout << number << " is even";
} else {
    cout << number << " is odd";  // This will be printed
}
```

#### **More Modulo Examples:**
```c++
cout << 10 % 3;  // Output: 1 (10 √∑ 3 = 3 remainder 1)
cout << 15 % 4;  // Output: 3 (15 √∑ 4 = 3 remainder 3)
cout << 20 % 5;  // Output: 0 (20 √∑ 5 = 4 remainder 0)
```

## ‚ö†Ô∏è Important Rules to Remember

### **1. Modulo Restrictions:**
The % operator cannot be used with floating-point numbers (float, double)

```c++
// ‚ùå This will cause an ERROR:
float a = 5.5;
float b = 2.2;
float result = a % b;  // COMPILE ERROR!

// ‚úÖ Correct way for floating-point remainder:
#include <cmath>
float result = fmod(a, b);  // Use fmod() function
```

### **2. Division by Zero:**
**Never divide by zero!** It will crash your program.

```c++
int a = 10;
int b = 0;
// int result = a / b;  // ‚ùå This will crash your program!

// ‚úÖ Always check first:
if (b != 0) {
    int result = a / b;
    cout << "Result: " << result;
} else {
    cout << "Cannot divide by zero!";
}
```

### **3. Negative Numbers with Modulo:**
The result with negative numbers is compiler-dependent

```c++
cout << -10 % 3;   // Could be -1 or 2 (depends on compiler)
cout << 10 % -3;   // Could be 1 or -2 (depends on compiler)
```

##  Operator Precedence (Order of Operations)

Just like in math class, multiplication, division, and modulo are calculated before addition and subtraction

### **Precedence Order (Highest to Lowest):**
1. **`*` `/` `%`** (Multiplication, Division, Modulo) - **Same priority**
2. **`+` `-`** (Addition, Subtraction) - **Same priority**

### **Examples:**
```c++
int result1 = 2 + 3 * 4;        // = 2 + 12 = 14 (not 20!)
int result2 = 10 - 6 / 2;       // = 10 - 3 = 7 (not 2!)
int result3 = 5 * 2 + 3 * 4;    // = 10 + 12 = 22

// Use parentheses to change the order:
int result4 = (2 + 3) * 4;      // = 5 * 4 = 20
int result5 = (10 - 6) / 2;     // = 4 / 2 = 2
```

### **Left-to-Right Association:**
When operators have the same precedence, they are evaluated from left to right

```c++
int result = 20 / 4 * 2;  // = (20 / 4) * 2 = 5 * 2 = 10
// NOT 20 / (4 * 2) = 20 / 8 = 2
```

## üõ†Ô∏è Practical Examples

### **Example 1: Calculate Area and Perimeter**
```c++
#include <iostream>
using namespace std;

int main() {
    double length = 10.5;
    double width = 6.2;
    
    double area = length * width;
    double perimeter = 2 * (length + width);
    
    cout << "Rectangle dimensions: " << length << " x " << width << endl;
    cout << "Area: " << area << " square units" << endl;
    cout << "Perimeter: " << perimeter << " units" << endl;
    
    return 0;
}
```

### **Example 2: Temperature Converter**
```c++
#include <iostream>
using namespace std;

int main() {
    double celsius = 25.0;
    
    // Convert Celsius to Fahrenheit: F = (C * 9/5) + 32
    double fahrenheit = (celsius * 9.0 / 5.0) + 32.0;
    
    cout << celsius << "¬∞C = " << fahrenheit << "¬∞F" << endl;
    
    return 0;
}
```

### **Example 3: Check if Year is Leap Year**
```c++
#include <iostream>
using namespace std;

int main() {
    int year = 2024;
    
    // A year is leap if divisible by 4
    if (year % 4 == 0) {
        cout << year << " is a leap year!" << endl;
    } else {
        cout << year << " is not a leap year." << endl;
    }
    
    return 0;
}
```

##  Key Takeaways

1. **Five basic operators:** `+`, `-`, `*`, `/`, `%`
2. **Order matters:** Multiplication, division, and modulo before addition and subtraction
3. **Integer vs. floating-point division:** Be careful with data types!
4. **Modulo is your friend:** Great for finding remainders, checking even/odd, and cycling through values
5. **Always avoid division by zero:** Check your divisors!
6. **Use parentheses:** When in doubt, make your intentions clear with `()`

###  Pro Tips:
- Use **meaningful variable names** to make your code readable
- **Test edge cases** like division by zero
- **Use parentheses** to make complex expressions clear
- **Consider data types** carefully when doing division
