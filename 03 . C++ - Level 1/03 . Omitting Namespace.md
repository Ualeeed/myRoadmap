```
Omitting Namespace mean No std:: by using [ using namespace std; ]
```

- Example :

```c++
#include <iostream>
using namespace std;

int main()
{
 cout << "Ualid Ahlidou"  ;

return 0;
}
```

before :

```
std::cout <<"Ualid Ahlidou" << std::endl;
```

But now by using <span style="color:rgb(181, 75, 251)">using namespace</span>   :

```
cout << "Ualid Ahlidou" ;
```
