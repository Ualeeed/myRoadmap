## What are Data Types?

Data types in C++ define the type and size of data that variables can store. They tell the compiler how to interpret and manipulate the data in memory. C++ contains three sets of types: fundamental data types (built-in), compound data types, and user-defined types.

## Categories of Data Types in C++

### 1. Fundamental (Primitive) Data Types
Primitive data types are the fundamental data types in C++, such as ``int``, char, float, double, bool, etc. These are built directly into the language and don't require any header files.

### 2. Derived Data Types
Derived data types are built on top of primitive data types, such as arrays and pointers.

### 3. User-Defined Data Types
These include classes, structures, unions, and enums that programmers create.

## Type Modifiers

There are 4 type modifiers in C++: short, long, signed and unsigned. The signed modifier can be used for both positive and negative values, the unsigned modifier is used for positive values only, the long modifier increases the size of data types, and the short modifier reduces their size.

### Available Modifiers:
1. **`signed`** - Stores both positive and negative values (default for int)
2. **`unsigned`** - Stores only positive values (including zero)
3. **`short`** - Reduces storage size
4. **`long`** - Increases storage size

### Modifiable Data Types:
- `int`
- `char`
- `double` (only with long)

## Practical Examples

### Basic Declaration and Usage
```c++
#include <iostream>
#include <string>
using namespace std;

int main() {
    // Integer types
    int age = 25;
    unsigned int population = 1000000;
    short temperature = -15;
    long long distance = 384400000; // Distance to moon in meters
    
    // Floating-point types
    float price = 19.99f;
    double pi = 3.14159265359;
    
    // Character types
    char grade = 'A';
    unsigned char byte_value = 255;
    
    // Boolean type
    bool is_student = true;
    
    // String type
    string name = "Ahmed";
    
    cout << "Age: " << age << endl;
    cout << "Price: " << price << endl;
    cout << "Grade: " << grade << endl;
    cout << "Is student: " << is_student << endl;
    cout << "Name: " << name << endl;
    
    return 0;
}
```

### Finding Size of Data Types
```c++
#include <iostream>
using namespace std;

int main() {
    cout << "=== Data Type Sizes ===" << endl;
    cout << "bool: " << sizeof(bool) << " bytes" << endl;
    cout << "char: " << sizeof(char) << " bytes" << endl;
    cout << "short: " << sizeof(short) << " bytes" << endl;
    cout << "int: " << sizeof(int) << " bytes" << endl;
    cout << "long: " << sizeof(long) << " bytes" << endl;
    cout << "long long: " << sizeof(long long) << " bytes" << endl;
    cout << "float: " << sizeof(float) << " bytes" << endl;
    cout << "double: " << sizeof(double) << " bytes" << endl;
    cout << "long double: " << sizeof(long double) << " bytes" << endl;
    
    return 0;
}
```

## Memory Usage Tips

### When to Use Each Type:

1. **`short`**: When memory is limited and values are small (-32,768 to 32,767)
2. **`int`**: General-purpose integer (most commonly used)
3. **`long long`**: For very large numbers (like timestamps, IDs)
4. **`unsigned`**: When you know values will never be negative
5. **`float`**: When precision isn't critical (graphics, approximate calculations)
6. **`double`**: When you need high precision (scientific calculations)
7. **`char`**: For single characters or small integers (-127 to 127)
8. **`bool`**: For true/false conditions

### Memory Efficiency Example:
```c++
// Less memory efficient
int small_counter = 10;        // Uses 4 bytes for small value

// More memory efficient  
short small_counter = 10;      // Uses 2 bytes for same value

// For large arrays, this difference matters:
int large_array[1000];         // 4000 bytes
short small_array[1000];       // 2000 bytes (50% less memory!)
```

## Common Mistakes to Avoid

1. **Overflow**: Using values outside the type's range
   ```cpp
   unsigned char value = 300;  // ERROR: Max is 255
   ```

2. **Precision Loss**: 
   ```cpp
   float pi = 3.14159265359;   // Will lose precision
   double pi = 3.14159265359;  // Better precision
   ```

3. **Mixing Signed and Unsigned**:
   ```cpp
   int a = -5;
   unsigned int b = 3;
   if (a < b) // Dangerous comparison!
   ```
