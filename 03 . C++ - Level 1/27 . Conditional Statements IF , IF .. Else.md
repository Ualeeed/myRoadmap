#### **Conditional statements**
in C++ allow us to execute different blocks of code based on whether certain conditions are true or false. The if statement is used to specify a block of code to be executed if a condition is true. The flow of a program is managed using conditional statements in C++.

## Basic IF Statement

### Syntax
```cpp
 if (condition)
 {
  // Code to execute if condition is true
  }
```

**Important:** Note that if is in lowercase letters. Uppercase letters (If or IF) will generate an error.

### Flow Diagram
```
┌─────────────────┐
│     START       │
└─────────────────┘
         │
         ▼
┌─────────────────┐
│   Condition?    │
└─────────────────┘
         │
    ┌────┴───────────────┐
  TRUE                 FALSE
    ▼                    │
┌─────────────┐          │
│Execute IF   │   		 │
│   Block     │  		 │
└─────────────┘          │
         │               │
         ▼               ▼
        ┌─────────────────┐
        │Continue Program │
        └─────────────────┘
```

### Examples

#### Simple IF Statement
```cpp
#include <iostream>
using namespace std;

int main() {
    int x = 10;
    
    if (x > 5) {
        cout << "x is greater than 5" << endl;
    }
    
    cout << "Program continues..." << endl;
    return 0;
}
// Output: x is greater than 5
//         Program continues...
```

#### IF Statement with False Condition
```cpp
#include <iostream>
using namespace std;

int main() {
    int x = 3;
    
    if (x > 5) {
        cout << "This won't execute" << endl;
    }
    
    cout << "Program continues..." << endl;
    return 0;
}
// Output: Program continues...
```

## IF...ELSE Statement

### Syntax
```cpp
if (condition) {
    // Code to execute if condition is true
} else {
    // Code to execute if condition is false
}
```

### Flow Diagram
```
┌─────────────────┐
│     START       │
└─────────────────┘
         │
         ▼
┌─────────────────┐
│   Condition?    │
└─────────────────┘
         │
    ┌────┴────────────┐
   TRUE              FALSE
    ▼                 ▼
┌─────────────┐ ┌─────────────┐
│Execute IF   │ │Execute ELSE │
│   Block     │ │   Block     │
└─────────────┘ └─────────────┘
         │         │
         └────┬────┘
              ▼
     ┌─────────────────┐
     │Continue Program │
     └─────────────────┘
```

### Examples

#### Age Verification Example
```cpp
#include <iostream>
using namespace std;

int main() {
    int age = 20;
    
    if (age >= 18) {
        cout << "You are an adult" << endl;
    } else {
        cout << "You are a minor" << endl;
    }
    
    return 0;
}
// Output: You are an adult
```

## IF...ELSE IF...ELSE Ladder

The if else if statements are used when the user has to decide among multiple options. The C if statements are executed from the top down. As soon as one of the conditions controlling the if is true, the statement associated with that if is executed, and the rest of the C else-if ladder is bypassed.

### Syntax
```cpp
if (condition1) {
    // Code for condition1
} else if (condition2) {
    // Code for condition2
} else if (condition3) {
    // Code for condition3
} else {
    // Default code
}
```

### Example: Grade Calculator
```cpp
#include <iostream>
using namespace std;

int main() {
    int score = 85;
    
    if (score >= 90) {
        cout << "Grade: A" << endl;
    } else if (score >= 80) {
        cout << "Grade: B" << endl;
    } else if (score >= 70) {
        cout << "Grade: C" << endl;
    } else if (score >= 60) {
        cout << "Grade: D" << endl;
    } else {
        cout << "Grade: F" << endl;
    }
    
    return 0;
}
// Output: Grade: B
```

## Nested IF Statements

### Syntax
```cpp
if (condition1) {
    if (condition2) {
        // Code when both conditions are true
    } else {
        // Code when condition1 is true but condition2 is false
    }
} else {
    // Code when condition1 is false
}
```

### Example: Login Validation
```cpp
#include <iostream>
#include <string>
using namespace std;

int main() {
    string username = "admin";
    string password = "12345";
    bool isActive = true;
    
    if (username == "admin") {
        if (password == "12345") {
            if (isActive) {
                cout << "Login successful!" << endl;
            } else {
                cout << "Account is inactive" << endl;
            }
        } else {
            cout << "Wrong password" << endl;
        }
    } else {
        cout << "Username not found" << endl;
    }
    
    return 0;
}
```

## Ternary Operator

The ternary operator is best for simple, inline conditional assignments where readability is not compromised.

### Syntax
```cpp
 condition ? value_if_true : value_if_false
```

### Examples

#### Basic Ternary Operator
```cpp
#include <iostream>
using namespace std;

int main() {
    int age = 20;
    string status = (age >= 18) ? "Adult" : "Minor";
    cout << "Status: " << status << endl;
    
    // Equivalent IF-ELSE:
    // if (age >= 18) {
    //     status = "Adult";
    // } else {
    //     status = "Minor";
    // }
    
    return 0;
}
// Output: Status: Adult
```

#### Finding Maximum
```cpp
#include <iostream>
using namespace std;

int main() {
    int a = 15, b = 10;
    int max = (a > b) ? a : b;
    cout << "Maximum: " << max << endl;
    return 0;
}
// Output: Maximum: 15
```

## Logical Operators

### AND Operator (&&)
Both conditions must be true for the entire expression to be true.

```cpp
#include <iostream>
using namespace std;

int main() {
    int x = 10;
    
    if (x > 5 && x < 15) {
        cout << "x is between 5 and 15" << endl;
    }
    
    return 0;
}
// Output: x is between 5 and 15
```

### OR Operator (||)
At least one condition must be true for the entire expression to be true.

```cpp
#include <iostream>
using namespace std;

int main() {
    int day = 7;
    
    if (day == 6 || day == 7) {
        cout << "It's weekend!" << endl;
    }
    
    return 0;
}
// Output: It's weekend!
```

### NOT Operator (!)
Reverses the boolean value of the condition.

```cpp
#include <iostream>
using namespace std;

int main() {
    bool isRaining = false;
    
    if (!isRaining) {
        cout << "Let's go outside!" << endl;
    }
    
    return 0;
}
// Output: Let's go outside!
```

## Best Practices

Best practices include writing clear conditions, avoiding excessive nesting, and using switch for multiple constant-value comparisons to enhance readability and efficiency.

### 1. Clear and Readable Conditions
```cpp
// Good
if (age >= 18 && hasLicense && !isDrunk) {
    cout << "Can drive" << endl;
}

// Avoid
if (a >= 18 && b && !c) {
    cout << "Can drive" << endl;
}
```

### 2. Avoid Deep Nesting
```cpp
// Good - Early return pattern
if (username != "admin") {
    cout << "Access denied" << endl;
    return 1;
}

if (password != "12345") {
    cout << "Wrong password" << endl;
    return 1;
}

cout << "Login successful" << endl;

// Avoid - Deep nesting
if (username == "admin") {
    if (password == "12345") {
        if (isActive) {
            if (hasPermission) {
                // Too deeply nested
            }
        }
    }
}
```

### 3. Use Braces Even for Single Statements
```cpp
// Good
if (condition) {
    doSomething();
}

// Avoid (prone to errors)
if (condition)
    doSomething();
```

### 4. When to Use Ternary vs IF-ELSE

Use ternary operator when you want some output value from result and there are only 2 conditions. An if/else statement emphasizes the branching first and what's to be done is secondary, while a ternary operator emphasizes what's to be done over the selection of the values to do it with.

```cpp
// Good use of ternary - simple assignment
int max = (a > b) ? a : b;

// Better as IF-ELSE - complex logic
if (score >= 90) {
    grade = 'A';
    cout << "Excellent performance!" << endl;
    updateRecord();
} else {
    grade = 'B';
    cout << "Good job!" << endl;
}
```

## Complete Example: Student Management System

```cpp
#include <iostream>
#include <string>
using namespace std;

int main() {
    // Student information
    string name = "John Doe";
    int age = 20;
    double gpa = 3.7;
    bool isEnrolled = true;
    int credits = 45;
    
    cout << "=== Student Management System ===" << endl;
    cout << "Student: " << name << endl;
    cout << "Age: " << age << endl;
    cout << "GPA: " << gpa << endl;
    cout << "Credits: " << credits << endl << endl;
    
    // Check enrollment status
    if (!isEnrolled) {
        cout << "Student is not currently enrolled." << endl;
        return 0;
    }
    
    // Determine academic standing
    if (gpa >= 3.5) {
        cout << "Academic Standing: Dean's List" << endl;
    } else if (gpa >= 3.0) {
        cout << "Academic Standing: Good Standing" << endl;
    } else if (gpa >= 2.0) {
        cout << "Academic Standing: Academic Probation" << endl;
    } else {
        cout << "Academic Standing: Academic Suspension" << endl;
    }
    
    // Determine class level
    string classLevel;
    if (credits >= 90) {
        classLevel = "Senior";
    } else if (credits >= 60) {
        classLevel = "Junior";
    } else if (credits >= 30) {
        classLevel = "Sophomore";
    } else {
        classLevel = "Freshman";
    }
    cout << "Class Level: " << classLevel << endl;
    
    // Check graduation eligibility
    if (credits >= 120 && gpa >= 2.0) {
        cout << "Status: Eligible for graduation!" << endl;
    } else if (credits >= 120) {
        cout << "Status: Need to improve GPA for graduation" << endl;
    } else {
        int needed = 120 - credits;
        cout << "Status: Need " << needed << " more credits" << endl;
    }
    
    // Scholarship eligibility (using ternary operator)
    string scholarship = (gpa >= 3.8 && credits >= 30) ? "Eligible" : "Not Eligible";
    cout << "Scholarship: " << scholarship << endl;
    
    return 0;
}
```

## Summary

Conditional statements are fundamental to programming logic in C++. They allow your programs to make decisions and execute different paths based on various conditions. Master these concepts:

1. **Basic IF**: Execute code only when condition is true
2. **IF-ELSE**: Choose between two alternatives
3. **IF-ELSE IF-ELSE**: Handle multiple conditions in sequence
4. **Nested IF**: Handle complex decision trees
5. **Ternary Operator**: Concise conditional assignments
6. **Logical Operators**: Combine multiple conditions
7. **Comparison Operators**: Compare values effectively

Remember to write clear, readable code and choose the right conditional structure for each situation!
