This lesson demonstrates how to effectively use for loops with arrays in C++ programming. Arrays and for loops work together seamlessly to process collections of data efficiently.

## Key Concepts

### Array Basics
- **Arrays** are collections of elements of the same data type
- Elements are stored in contiguous memory locations
- Array indexing starts from 0
- Declaration: `datatype arrayName[size]`

### For Loop with Arrays
For loops are the most common way to iterate through arrays because:
- You can easily control the index with the loop counter
- The loop counter directly corresponds to array indices
- Perfect for processing each element sequentially

## Program Breakdown

### Core Functionality
Our program demonstrates four key operations:

1. **ReadArrayData()** - Input function that:
   - Asks user for the number of elements to enter
   - Uses a for loop to read each number into the array
   - Stores the count in a reference parameter

2. **PrintArrayData()** - Output function that:
   - Uses a for loop to display each array element
   - Shows both the position and value of each element

3. **CalculateArraySum()** - Processing function that:
   - Iterates through the array using a for loop
   - Accumulates the sum of all elements
   - Returns the total sum

4. **CalculateArrayAverage()** - Calculation function that:
   - Reuses the sum function
   - Performs type casting to float for precise division
   - Returns the average value

## Complete Code Example

```cpp
#include <iostream>
#include <string>
using namespace std;

void ReadArrayData(int Array[100], int &Length)
{
    cout << "How many numbers do you want to enter? (1 to 100): ";
    cin >> Length;
    
    // Validate input range
    while (Length < 1 || Length > 100) {
        cout << "Please enter a number between 1 and 100: ";
        cin >> Length;
    }

    for (int i = 0; i <= Length - 1; i++)
    {
        cout << "Enter Number " << i + 1 << ": ";
        cin >> Array[i];
    }
}

void PrintArrayData(int Array[100], int Length)
{
    cout << "\n--- Your Numbers ---\n";
    for (int i = 0; i <= Length - 1; i++)
    {
        cout << "Number [" << i + 1 << "]: " << Array[i] << endl;
    }
}

int CalculateArraySum(int Array[100], int Length)
{
    int Sum = 0;
    
    for (int i = 0; i <= Length - 1; i++)
    {
        Sum += Array[i];
    }
    
    return Sum;
}

float CalculateArrayAverage(int Array[100], int Length)
{
    return (float)CalculateArraySum(Array, Length) / Length;
}

int main()
{
    int Array[100], Length = 0;
    
    ReadArrayData(Array, Length);
    PrintArrayData(Array, Length);
    
    cout << "\n--- Results ---\n";
    cout << "Sum = " << CalculateArraySum(Array, Length) << endl;
    cout << "Average = " << CalculateArrayAverage(Array, Length) << endl;
    
    return 0;
}
```

## Alternative For Loop Patterns

### Different Loop Styles
```cpp
// Standard style (used in main example)
for (int i = 0; i <= Length - 1; i++)

// More common style
for (int i = 0; i < Length; i++)

// Reverse iteration
for (int i = Length - 1; i >= 0; i--)

// Step by 2
for (int i = 0; i < Length; i += 2)
```

## Additional Array Operations

### Finding Maximum Element
```cpp
int FindMaxElement(int Array[], int Length)
{
    int max = Array[0];
    for (int i = 1; i < Length; i++)
    {
        if (Array[i] > max)
            max = Array[i];
    }
    return max;
}
```

### Counting Specific Elements
```cpp
int CountOccurrences(int Array[], int Length, int target)
{
    int count = 0;
    for (int i = 0; i < Length; i++)
    {
        if (Array[i] == target)
            count++;
    }
    return count;
}
```

## Best Practices

### Memory Management
- Use appropriate array sizes
- Always validate array bounds
- Initialize arrays when possible

### Loop Optimization
- Use `i < Length` instead of `i <= Length - 1` for clarity
- Consider using range-based for loops in C++11 and later
- Avoid modifying the loop variable inside the loop body

### Function Design
- Pass arrays by reference to avoid copying
- Use const for read-only array parameters
- Always pass the array size as a separate parameter

## Common Pitfalls to Avoid

1. **Array Index Out of Bounds**: Always ensure loop conditions don't exceed array size
2. **Uninitialized Arrays**: Initialize arrays before use to avoid garbage values
3. **Off-by-One Errors**: Be careful with loop termination conditions
4. **Integer Division**: Remember to cast to float for decimal averages
