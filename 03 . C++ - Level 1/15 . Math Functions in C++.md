The C++ `<cmath>` header file declares a set of functions to perform mathematical operations such as: sqrt() to calculate the square root, log() to find natural logarithm of a number etc.

## Prerequisites
First we need to include the `cmath` library to use mathematical functions:
```cpp
#include <cmath>
```

## Basic Mathematical Functions

### 1. **Square Root Functions**

#### `sqrt()` - Square Root
```cpp
#include <iostream>
#include <cmath>
using namespace std;

int main() {
    double x = 64;
    cout << sqrt(x) << endl;    // gives 8
    cout << sqrt(50) << endl;   // gives 7.07107
    cout << sqrt(25.0) << endl; // gives 5
    return 0;
}
```

### 2. **Rounding Functions**

#### `round()` - Round to Nearest Integer
```cpp
#include <iostream>
#include <cmath>
using namespace std;

int main() {
    cout << round(2.4) << endl; // gives 2
    cout << round(2.5) << endl; // gives 3
    cout << round(2.7) << endl; // gives 3
    cout << round(-2.5) << endl; // gives -2
    
    cout << sqrt(50) << endl;        // gives 7.07107
    cout << round(sqrt(50)) << endl; // gives 7
    return 0;
}
```

#### `ceil()` & `floor()` - Ceiling and Floor
**Ceiling (السقف)**: Rounds UP to the nearest integer  
**Floor (الارض)**: Rounds DOWN to the nearest integer

```cpp
#include <iostream>
#include <cmath>
using namespace std;

int main() {
    // Positive numbers
    cout << ceil(2.1) << endl;  // gives 3
    cout << floor(2.9) << endl; // gives 2
    
    // Negative numbers
    cout << ceil(-2.9) << endl;  // gives -2
    cout << floor(-2.1) << endl; // gives -3
    
    return 0;
}
```

#### `trunc()` - Truncate Decimal Part
```cpp
#include <iostream>
#include <cmath>
using namespace std;

int main() {
    cout << trunc(2.9) << endl;  // gives 2
    cout << trunc(-2.9) << endl; // gives -2
    return 0;
}
```

### 3. **Power Functions**

#### `pow()` - Power/Exponentiation
```cpp
#include <iostream>
#include <cmath>
using namespace std;

int main() {
    int x = 2;
    int y = 4;
    
    cout << pow(x, y) << endl;   // gives 16 (2^4)
    cout << pow(4, 3) << endl;   // gives 64 (4^3)
    cout << pow(2.5, 2) << endl; // gives 6.25
    
    return 0;
}
```

#### `exp()` - Exponential Function (e^x)
```cpp
#include <iostream>
#include <cmath>
using namespace std;

int main() {
    cout << exp(1) << endl;  // gives 2.71828 (e^1)
    cout << exp(2) << endl;  // gives 7.38906 (e^2)
    return 0;
}
```

### 4. **Absolute Value Functions**

#### `abs()` - Absolute Value for Integers
#### `fabs()` - Absolute Value for Floating-Point Numbers
```cpp
#include <iostream>
#include <cmath>
using namespace std;

int main() {
    // For integers
    cout << abs(-10) << endl;   // gives 10
    cout << abs(10) << endl;    // gives 10
    
    // For floating-point numbers
    cout << fabs(-3.14) << endl; // gives 3.14
    cout << fabs(3.14) << endl;  // gives 3.14
    
    return 0;
}
```

## Logarithmic Functions

### `log()` - Natural Logarithm (ln)
```cpp
#include <iostream>
#include <cmath>
using namespace std;

int main() {
    cout << log(1) << endl;      // gives 0
    cout << log(2.71828) << endl; // gives ~1 (ln(e))
    cout << log(10) << endl;     // gives 2.30259
    return 0;
}
```

### `log10()` - Base-10 Logarithm
```cpp
#include <iostream>
#include <cmath>
using namespace std;

int main() {
    cout << log10(1) << endl;     // gives 0
    cout << log10(10) << endl;    // gives 1
    cout << log10(100) << endl;   // gives 2
    cout << log10(1000) << endl;  // gives 3
    return 0;
}
```

### `log2()` - Base-2 Logarithm
```cpp
#include <iostream>
#include <cmath>
using namespace std;

int main() {
    cout << log2(1) << endl;   // gives 0
    cout << log2(2) << endl;   // gives 1
    cout << log2(8) << endl;   // gives 3
    cout << log2(16) << endl;  // gives 4
    return 0;
}
```

## Trigonometric Functions

### Basic Trigonometric Functions
```cpp
#include <iostream>
#include <cmath>
using namespace std;

int main() {
    double angle = 3.14159 / 4; // 45 degrees in radians
    
    cout << sin(angle) << endl;  // gives ~0.707
    cout << cos(angle) << endl;  // gives ~0.707  
    cout << tan(angle) << endl;  // gives ~1
    
    return 0;
}
```

### Inverse Trigonometric Functions
```cpp
#include <iostream>
#include <cmath>
using namespace std;

int main() {
    cout << asin(0.5) << endl;   // gives ~0.524 (30° in radians)
    cout << acos(0.5) << endl;   // gives ~1.047 (60° in radians)
    cout << atan(1) << endl;     // gives ~0.785 (45° in radians)
    
    return 0;
}
```

## Advanced Functions

### `fmod()` - Floating Point Remainder
```cpp
#include <iostream>
#include <cmath>
using namespace std;

int main() {
    cout << fmod(7.0, 3.0) << endl;  // gives 1.0 (7 % 3)
    cout << fmod(8.5, 2.5) << endl;  // gives 1.0
    return 0;
}
```

### `fmax()` and `fmin()` - Maximum and Minimum
```cpp
#include <iostream>
#include <cmath>
using namespace std;

int main() {
    cout << fmax(3.5, 2.8) << endl; // gives 3.5
    cout << fmin(3.5, 2.8) << endl; // gives 2.8
    return 0;
}
```

### `modf()` - Split into Integer and Fractional Parts
```cpp
#include <iostream>
#include <cmath>
using namespace std;

int main() {
    double intpart;
    double fractpart = modf(3.14159, &intpart);
    
    cout << "Integer part: " << intpart << endl;   // gives 3
    cout << "Fractional part: " << fractpart << endl; // gives 0.14159
    
    return 0;
}
```

## Important Notes

- All trigonometric functions work with **radians**, not degrees
- To convert degrees to radians: `radians = degrees * (π / 180)`
- To convert radians to degrees: `degrees = radians * (180 / π)`
- Use `fabs()` for floating-point numbers and `abs()` for integers
- The `cmath` library is essential for mathematical computations in C++
