### Increment Operator (`++`)

- **Meaning**: Increases a variable’s value by **1**.
    
- **Forms**:
    
    - **Pre-increment**: `++x` → increments first, then uses the new value.
        
    - **Post-increment**: `x++` → uses the current value, then increments.

- Example : 

```c++
int x = 5;
cout << ++x;  // prints 6 (increment first, then use)
cout << x++;  // prints 6 (use first, then increment → x becomes 7 afterward)
```

### Decrement Operator (`--`)

- **Meaning**: Decreases a variable’s value by **1**.
    
- **Forms**:
    
    - **Pre-decrement**: `--x` → decrements first, then uses the new value.
        
    - **Post-decrement**: `x--` → uses the current value, then decrements.
- Example : 

```c++
int y = 5;
cout << --y;  // prints 4 (decrement first, then use)
cout << y--;  // prints 4 (use first, then decrement → y becomes 3 afterward)
```

| Operations | Description                            | Ex  |
| ---------- | -------------------------------------- | --- |
| ++         | A++ -> A = A + 1<br>++A -> A = A + 1   |     |
| --         | B--  -> B = B - 1 <br>-- B -> B = B -1 |     |


Ex : 

```c++

#include <iostream>
using namespace std; 

int main()
{

    int A = 10, B = 20;

    A++; // Same as A = A + 1
    B--; //Same as B = B - 1


 cout << "A = " << A <<endl;
 cout << "B = " << B <<endl;
 

return 0;


}

```
