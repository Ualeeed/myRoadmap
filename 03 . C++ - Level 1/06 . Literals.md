#  Literals in C++

<mark style="background: #FFB86CA6;">Literals</mark> are <mark style="background: #FF5582A6;">fixed values</mark> that are directly embedded in the source code and represent constant data. They are fundamental elements used to represent fixed values including numbers, characters, strings, and more. Unlike variables, literals cannot be modified during program execution.

**Example:**
```c++
int age = 25;        // 25 is an integer literal
float pi = 3.14f;    // 3.14f is a floating-point literal
char grade = 'A';    // 'A' is a character literal
string name = "John"; // "John" is a string literal
```

There are six major categories of literals in C++: 
```
integer, character, floating-point, string, boolean, and pointer. 
```

##### **Let 's explore each type in detail:**

### **01 . Integer Literals**

Integer literals represent whole numbers without fractional parts. They can be expressed in different number bases:

#### **Base Representations:**

| **Type** | **Base** | **Prefix** | **Example** | **Value** |
|----------|----------|------------|-------------|-----------|
| **Decimal** | Base 10 | None | `42` | 42 |
| **Octal** | Base 8 | `0` | `052` | 42 |
| **Hexadecimal** | Base 16 | `0x` or `0X` | `0x2A` | 42 |
| **Binary** | Base 2 | `0b` or `0B` | `0b101010` | 42 |

#### **Integer Suffixes:**

| **Suffix** | **Type** | **Example** |
|------------|----------|-------------|
| `L` or `l` | long | `42L` |
| `U` or `u` | unsigned | `42U` |
| `LL` or `ll` | long long | `42LL` |
| `UL` | unsigned long | `42UL` |

**Example Code:**
```c++
int decimal = 42;        // Decimal literal
int octal = 052;         // Octal literal (starts with 0)
int hex = 0x2A;          // Hexadecimal literal (starts with 0x)
int binary = 0b101010;   // Binary literal (starts with 0b)
long longNum = 42L;      // Long integer literal
unsigned int unsignedNum = 42U; // Unsigned integer literal
```

---

### **02 . Floating-Point Literals:**

Floating-point literals represent numbers with fractional or decimal parts. They can be single-precision, double-precision, or extended precision.

#### **Floating-Point Types:**

| **Suffix** | **Type** | **Example** | **Precision** |
|------------|----------|-------------|---------------|
| `f` or `F` | float | `3.14f` | Single |
| None or `d` | double | `3.14` | Double (default) |
| `l` or `L` | long double | `3.14L` | Extended |

#### **Scientific Notation:**
For very large or small numbers:
```c++
double large = 1.23e6;    // 1.23 Ã— 10^6 = 1230000
double small = 1.23e-6;   // 1.23 Ã— 10^-6 = 0.00000123
```

**Example Code:**
```c++
float singlePrecision = 3.14f;
double doublePrecision = 3.141592653589793;
long double extended = 3.141592653589793238L;
double scientific = 6.022e23;  // Avogadro's number
```

---

### **03 . Character Literals:**

Character literals represent single characters enclosed in single quotes. They use the ASCII character set with 128 possible character literals.

**Examples:**
```c++
char letter = 'A';
char digit = '5';
char symbol = '@';
char newline = '\n';  // Escape sequence
```

---

### **04 . String Literals:**

String literals are sequences of characters enclosed in double quotes, used to represent text or character sequences.

**Examples:**
```c++
const char* message = "Hello, World!";
string greeting = "Welcome to C++";
const char* empty = "";           // Empty string
const char* multiline = "Line 1\n"
                       "Line 2\n"
                       "Line 3";   // Concatenated string literals
```

---

### **05 . Boolean Literals:**

Boolean literals represent truth values with only two possible values: true and false. They are used in Boolean expressions and logical operations.

**Examples:**
```c++
bool isActive = true;
bool isComplete = false;

if (isActive) {
    cout << "System is running";
}
```

---

### **06 . Pointer Literals:**

C++ introduces the ``nullptr`` literal to specify a zero-initialized pointer. This is the modern way to represent null pointers.

**Examples:**
```c++
int* ptr = nullptr;        // Modern C++ (preferred)
int* oldPtr = NULL;        // Legacy C style (avoid in modern C++)
```

---

## **07 . Escape Sequences:**

Escape sequences are character combinations consisting of a backslash ( \ ) followed by a letter or combination of digits. They start with a backslash ( \ ) and are used inside strings or characters to control formatting or show special symbols.

### **Common Escape Sequences:**

| **Escape Sequence** | **Meaning** | **ASCII Value** |
|-------------------|-------------|-----------------|
| `\n` | Newline | 10 |
| `\t` | Horizontal tab | 9 |
| `\r` | Carriage return | 13 |
| `\\` | Backslash | 92 |
| `\"` | Double quote | 34 |
| `\'` | Single quote | 39 |
| `\a` | Alert/Bell | 7 |
| `\b` | Backspace | 8 |
| `\f` | Form feed | 12 |
| `\v` | Vertical tab | 11 |
| `\0` | Null character | 0 |

### **Numeric Escape Sequences:**

| **Type** | **Format** | **Example** | **Description** |
|----------|------------|-------------|-----------------|
| **Octal** | `\ooo` | `\101` | Character with octal value (A) |
| **Hexadecimal** | `\xhh` | `\x41` | Character with hex value (A) |
| **Unicode** | `\uxxxx` | `\u0041` | Unicode character (A) |

**Example Usage:**
```c++
cout << "Hello\tWorld\n";           // Tab and newline
cout << "She said \"Hello!\"";      // Embedded quotes
cout << "Path: C:\\Users\\Name";    // Backslashes in path
cout << "Bell sound: \a";           // Alert sound
```

##  Best Practices

### **1. Use Appropriate Suffixes**
```c++
float radius = 5.0f;        // Use 'f' for float literals
long distance = 1000000L;   // Use 'L' for long literals
```

### **2. Use nullptr Instead of NULL**
```c++
int* ptr = nullptr;         // Modern C++ style âœ…
int* oldPtr = NULL;         // Avoid in modern code âŒ
```
 
### **3. Choose Appropriate Number Base**
```c++
int permissions = 0755;     // Octal for file permissions
int color = 0xFF0000;       // Hex for colors (red)
int flags = 0b11010011;     // Binary for bit patterns
```

### **4. Be Mindful of Type Deduction**
```c++
auto val1 = 42;     // int
auto val2 = 42L;    // long
auto val3 = 42.0;   // double
auto val4 = 42.0f;  // float
```

##  Key Takeaways

- **Literals are compile-time constants** that cannot be modified during program execution
- **Different bases** (decimal, octal, hex, binary) provide flexibility for different use cases
- **Suffixes determine the exact type** of integer and floating-point literals
- **Escape sequences enable special characters** in strings and character literals
- **Use modern C++ features** like `nullptr` instead of legacy alternatives
- **Choose appropriate literal types** to match your data requirements and improve code clarity

Understanding literals is fundamental to C++ programming as they form the basis for initializing variables and representing constant values throughout your code.
