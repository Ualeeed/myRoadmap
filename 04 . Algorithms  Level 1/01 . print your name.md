# Requirement : 

- Write a program that print your name on the screen.
# Solution :  


```c++
#include <iostream>
using namespace std;

void PrintName(string Name)
{
	cout << " \n Your name is : " << Name << endl;
}


int main()
{

	PrintName("Ualid");


	return 0;
}

```

# Explanation

- **Headers/namespace**: Includes `<iostream>` and brings `std` into scope.
- **Function**: `PrintName(string Name)` prints a label plus the provided name via `cout`.
- **Entry point**: `main` calls `PrintName("Ualid")` and returns 0.

Expected output:

```
Your name is : Ualid
```

Notes:
- **Prefer qualifiers**: Use `std::cout`/`std::string` instead of `using namespace std;` in larger programs.
- **Input option**: Read a name with `std::getline(std::cin, name)` and pass it to `PrintName`.
