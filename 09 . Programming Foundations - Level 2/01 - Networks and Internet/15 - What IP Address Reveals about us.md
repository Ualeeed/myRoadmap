## ğŸŒ 2. Internet Service Provider (ISP)

### What's Revealed:

Your IP address clearly identifies which Internet Service Provider you're using. This information is public and easily accessible.

**ISP Information Includes:**
- Company name (e.g., AT&T, Comcast, Orange, Maroc Telecom)

- **Bandwidth Management**: ISPs can manage traffic based on usage
- **Legal Requests**: Law enforcement can request user info from ISP

## ğŸ›¡ï¸ 6. Privacy Protection Methods

### How to Protect Your Privacy:

**1. Use a VPN (Virtual Private Network)**
- Hides your real IP address
- Encrypts your internet traffic
- Makes you appear in different location
- [[16 - What is VPN]] - Learn more about VPNs

**2. Use Tor Browser**
- Routes traffic through multiple nodes
- Provides strong anonymity
- Free and open-source

**3. Use Proxy Servers**
- Intermediary server masks your IP
- Less secure than VPN
- Often free but slower

**4. Dynamic IP Addresses**
- Your IP changes periodically
- Harder to track long-term
- Most home users have dynamic IPs

**5. Browser Privacy Features**
- Private/Incognito mode (limited protection)
- Disable third-party cookies
- Use privacy-focused browsers
- Install ad blockers

## ğŸ“Š Dynamic vs Static IP Addresses

### Understanding IP Types:

| Feature | Dynamic IP | Static IP |
|---------|-----------|-----------|
| **Changes** | Regularly (days/weeks) | Never (unless manually changed) |
| **Typical Users** | Home users | Businesses, servers |
| **Tracking** | Harder to track long-term | Easy to track |
| **Cost** | Usually free | Often extra charge |
| **Use Case** | General internet use | Web servers, remote access |

**Impact on Privacy:**
- **Dynamic IPs**: Provide some natural privacy as they change
- **Static IPs**: More vulnerable to long-term tracking
